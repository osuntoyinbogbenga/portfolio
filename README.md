# Gbenga Osuntoyinbo — Portfolio

Personal portfolio website. Dark amber/orange theme with Three.js 3D hero, GSAP scroll animations, experience timeline, and a separate contact page.

## File Structure

portfolio/
├── index.html      # Main page (Hero, About, Skills, Projects, Experience)
├── contact.html    # Separate contact page with form
├── style.css       # All styles
├── main.js         # Three.js, GSAP, animations, interactions
└── README.md

## Run Locally

npx serve .

Then open http://localhost:3000

## Contact Form Setup

The contact form uses Formspree (free).

## Push to GitHub
 https://github.com/osuntoyinbogbenga/portfolio.git


## Customisation

Accent color: change --accent in :root in style.css
  Current: #f97316 (orange)

Fonts: Syne (headings) + Instrument Serif (italic accents) + DM Mono (labels)
